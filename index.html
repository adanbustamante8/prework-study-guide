HTML

  • HTML stands for Hyper Tex Markup language
  • HTML describes the structure of a web page
  • HTML is the standard markup language for creating webpages
  • The head element contains information about the webpage
  • The body elements represents the visible content shown to the user
  • HTML consists of a series of elements
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph" etc

CSS

  • Cascading Style Sheets (CSS) is a programming language used by developers to define how webpage content is presented to users
  • CSS defines color, aesthetics, visual layouyt and organization
  • Use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • 3 ways to style a wepage using CSS, inline CSS, internal CSS, and external CSS
  • Inline CSS: contains the CSS property in the body section attached with the element known as inline CSS
  • Internal CSS:The CSS ruleset should be within the HTML file in the head section i.e the CSS is embedded within the HTML file
  • External CSS: contains a seperate CSS file that contains only style property with the help of tag attributes

Git

  • Git is a web based interface that uses Git
  • Version control software that allows people to work on a program with out getting in each others way
  • Repository: a folder in which all files and their version histories are stored
  • Branch:a workspace in which you can make changes that wont affect the live site
  • Pull Request: the way to ask for changes made to a branch to be merged into another branch
  • Merge: after a pull request is approved, the commit will be pulled in from one branch to another and then,deployed on the live site
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git branch [branch name]: create a new branch at the current commit
  • git log: shows all commits in the current branch's history

JavaScript

  • JavaScript is a high levelk dynamic programming language that allows users to interact with websites
  • Using JS, you can power your websites with effects, interactivity, dtata communication and much more
  • Each line of JS code ends with a semicolon
  • Variables are containers that store data in our code
  • for loops use the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
