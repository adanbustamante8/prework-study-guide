Git

  • Git is a web based interface that uses Git
  • Version control software that allows people to work on a program with out getting in each others way
  • Repository: a folder in which all files and their version histories are stored
  • Branch:a workspace in which you can make changes that wont affect the live site
  • Pull Request: the way to ask for changes made to a branch to be merged into another branch
  • Merge: after a pull request is approved, the commit will be pulled in from one branch to another and then,deployed on the live site
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git branch [branch name]: create a new branch at the current commit
  • git log: shows all commits in the current branch's history
